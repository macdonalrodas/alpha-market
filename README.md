## DESCRIPCIÓN DEL PROYECTO

Alpha Market es una plataforma de e-commerce diseñada bajo una **arquitectura modular**, donde cada módulo tiene una responsabilidad bien definida y se comunica con los demás a través de interfaces claras. El proyecto aplica los principios fundamentales de la Ingeniería de Software: bajo acoplamiento, alta cohesión, mantenibilidad y escalabilidad.

---

## MÓDULOS DEL SISTEMA

| Módulo | Descripción |
|---|---|
|  Registro de Usuarios | Creación de cuentas con validación de correo y activación por token UUID |
|  Autenticación | Login seguro con JWT, bloqueo tras 5 intentos y recuperación de contraseña |
|  Gestión de Productos | Catálogo administrable con imágenes, categorías y precios |
|  Carrito de Compras | Gestión de artículos con cálculo automático de subtotal, IVA (16%) y descuentos |
|  Gestión de Almacenamiento | Control de inventario con alertas de stock bajo |
|  Procesamiento de Pagos | Integración simulada con pasarela de pago externa vía HTTPS |
